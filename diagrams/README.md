# Diagrams, B06

Architecture drawings, decision trees, and cover art for this article.

## Files

- `cover.png` Hashnode cover (1600×840). Generate with `tools/generate_cover.py`.
- `architecture.mmd` Mermaid reference architecture (authoritative).
- Other `.mmd` / `.excalidraw` / `.svg` as the article develops.

## Conventions

- Keep source (`.mmd`, `.excalidraw`) under version control. Exports (`.png`) go in `exports/` and are git-ignored.
- All diagrams use the palette defined in `shared/diagrams/palette.md`.
